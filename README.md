  - 4.3-Objetivos operativos
  - 4.4-Acciones planificadas
  - 4.5-Recursos
  - 4.6-Implantación del plan
- **5.Organización:**

  - 5.2-Organigrama
  - 5.3-RSC
- **6.Dirección:**
  - 6.1-Propiedad de la empresa
  - 6.2-Gobierno de la empresa

  - 6.4-Producción

  - 6.7-Recursos humanos (No disponible)
